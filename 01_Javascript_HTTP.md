= Network 탭에서 확인할 수 있음

* HTTP(프로토콜)라는 통신 방식으로 정보를 주고 받는다.
* Request Headers, Response Headers가 그 방식.
* 문제가 생겼을 경우, 어떤 Header의 어떤 내용이 어떻게 되어있길래 내 웹브라우저가 이렇게 반응하는 것이다를 알아야 해결할 수 있는 것.
